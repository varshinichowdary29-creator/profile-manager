# FirstCry Intellitots - Frontend Client

A React SPA web client powered by Vite and TailwindCSS for the Student Profile Manager application.

## Getting Started

### Setup
1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Environment Variables:**
   Copy `.env.example` to `.env` and set the backend API endpoint URL:
   ```bash
   cp .env.example .env
   ```

3. **Run Dev Client:**
   Start the development server:
   ```bash
   npm run dev
   ```

4. **Build Production Bundle:**
   Compile Vite assets:
   ```bash
   npm run build
   ```

## Key Technologies
* **React 18**
* **Vite** (Compilation and bundling)
* **TailwindCSS** (Vanilla stylesheet rules)
* **Framer Motion** (Subtle micro-animations)
* **Chart.js** (Renders radar milestone charts)
* **Lucide Icons**
